# ShitBlocker
Removes shit from the Internet

## Motivation

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```

### Run unit tests
```
npm run test:unit
```

### Compiles extension scripts
```
webpack
```
Run from extensionScripts folder
